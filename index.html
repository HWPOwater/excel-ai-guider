Раздел 1

ИИ прямо в Excel

Нативные интеграции, которые работают внутри привычного интерфейса. Никаких копипастов — ИИ видит вашу книгу целиком.

⚡ Что изменилось в 2025–2026

ИИ больше не просто «генерирует формулы». Claude for Excel, ChatGPT for Excel и Copilot понимают архитектуру всей книги, отслеживают связи между листами и работают с финансовыми источниками данных (LSEG, Moody's, S&P Capital IQ). Разница с 2023-м — как между калькулятором и старшим аналитиком.

🔮
Claude for Excel
Anthropic · Native Add-in
Pro $20/мес Лучший выбор

Нативный сайдбар с полным пониманием структуры книги. Читает все листы, прослеживает зависимости ячеек, объясняет каждое изменение. Подключён к финансовым источникам LSEG, Moody's и S&P Capital IQ.

  • Понимает архитектуру всей книги — не одного листа
  • Отслеживает что и где изменил, показывает диффы
  • Связка с финансовыми данными прямо в ячейках
  • Вариант анализа план-факт по 3+ файлам через Cowork
  • Защита от prompt-injection из ячеек (важно!)
Шаг 1 — Установка Add-in (Windows 11)

Откройте Excel → вкладка Главная (Home) на ленте → кнопка Add-ins → в строке поиска наберите «Claude» → выберите «Claude by Anthropic for Excel» → нажмите Add. Альтернативно: перейдите в Microsoft AppSource по кнопке ниже → «Get it now» → Excel откроется автоматически с уже активным Add-in.

Шаг 2 — Вход и активация

После установки в правой части экрана появится боковая панель Claude. Нажмите Sign in → войдите через аккаунт Anthropic (claude.ai). Требуется подписка Pro ($20/мес) или выше. Быстрый вызов панели в любой момент: Ctrl + Alt + C.

Шаг 3 — Правильный подход к работе

Не используйте как поисковик по формулам. Опишите задачу целиком: «У меня бюджет на листах Q1–Q4, нужно план-факт за год с отклонениями в % и выделением красным > 15%». Claude видит всю книгу — ссылайтесь на конкретные листы и ячейки.

Шаг 4 — Готовый промпт для старта
Первый промпт в новой книге
Ты видишь книгу Excel. Лист «Факт» — данные выгрузки из 1С. Лист «Бюджет» — плановые показатели. Создай на новом листе «Анализ» сводную таблицу план-факт по статьям ДДС с колонками: план, факт, абс. откл., % откл. Ячейки с отклонением >15% выдели красным, <5% — зелёным. Все формулы должны быть ссылками, без hardcode.
⚠️ Важно: безопасность

Никогда не открывайте через Claude файлы, полученные от внешних контрагентов без проверки. В ячейках могут быть скрытые инструкции (prompt injection — уязвимость CellShock). Anthropic добавила защиту, но правило остаётся: только доверенные файлы.

Шаг 2 — Вход в аккаунт

В боковой панели нажмите Sign in with OpenAI. Требуется подписка ChatGPT Plus или Pro. Сейчас доступно в бета-версии для пользователей из США, Канады, Австралии — для остальных стран используйте веб-версию ChatGPT с загрузкой файла.

Шаг 3 — Сильный кейс: текстовый отчёт

ChatGPT for Excel особенно силён в создании управленческих нарративов по данным таблицы. Идеально для еженедельных резюме и отчётов для руководства.

Промпт для управленческого резюме
На листе «Данные» — финансовые показатели компании за Q1 2025. Напиши управленческое резюме (300 слов) для CFO: ключевые тренды выручки, 3 проблемных направления, 2 позитивных момента, рекомендации. Стиль: профессиональный, без воды.
Claude vs ChatGPT — когда что выбрать

Claude — глубокое понимание структуры книги, аудит формул, план-факт по нескольким листам.
ChatGPT — текстовые выводы по данным, шаблоны, агентное выполнение цепочки задач.

Шаг 1 — Проверьте лицензию (Windows 11)

Откройте Excel → Файл → Учётная запись → Сведения о продукте. Убедитесь что у вас Microsoft 365 (не Office 2019/2021). Copilot в Excel требует отдельную лицензию: Copilot Pro ($20/мес) для частных лиц или Microsoft 365 Copilot для бизнеса.

Шаг 2 — Активация

После покупки лицензии: Excel → Файл → Учётная запись → Обновить лицензию. Перезапустите Excel. Кнопка Copilot появится на вкладке Главная в правой части ленты. Быстрый вызов: Alt + H → FX (через клавишные подсказки на ленте).

Шаг 3 — Если Copilot не отображается

Excel → Файл → Параметры → Центр управления безопасностью → Параметры конфиденциальности → включите «Все подключённые функции». Если управляется организацией — обратитесь к IT-администратору.

Google Gemini
Google · Sheets + чат с Excel
Free tier · лимит/день Gemini 3.1 Pro

Нативно встроен в Google Sheets — кнопка «Ask Gemini» прямо в интерфейсе. Для Excel — загружаете файл в чат и работаете. Функция =AI() прямо в ячейках Sheets. Бесплатный тариф даёт доступ к Gemini 3.0 / 3.1 Pro с дневным лимитом запросов — для большинства офисных задач хватает.

  • Нативный сайдбар в Google Sheets — бесплатно
  • Формулы, анализ, визуализация по описанию задачи
  • Fill with Gemini — автозаполнение таблиц из веба
  • Excel-файлы: загружаете в чат → получаете анализ
  • Экспорт результатов обратно в Sheets или Excel
Шаг 1 — Два способа работы (Windows 11)

Способ А — через Google Sheets (лучший): Откройте браузер → перейдите на sheets.google.com → откройте или создайте таблицу → нажмите иконку ✨ Ask Gemini в правом верхнем углу → сайдбар открыт. Бесплатно для всех аккаунтов Google.

Способ Б — через чат с Excel-файлом: Перейдите на gemini.google.com → нажмите скрепку → загрузите .xlsx файл → задавайте вопросы.

Шаг 2 — Функция =AI() в ячейках

В Google Sheets можно использовать =AI(«запрос», A2) прямо в ячейке. Например: =AI("категоризируй этот расход: Аренда/Зарплата/ИТ/Прочее", A2) — классифицирует весь столбец автоматически. Доступно в Google Workspace или Google AI Pro.

Шаг 3 — Промпт для анализа финансов
Промпт в Gemini Sheets / чате
Это таблица с транзакциями за квартал (столбцы: Дата, Контрагент, Сумма, Категория). Сделай: 1) Сводку по категориям — сумма и % от общего; 2) Топ-5 контрагентов по объёму; 3) Помесячную динамику расходов; 4) Вырви аномальные суммы (больше 3σ). Результаты вставь в новый лист «Анализ».
⚠️ Важно: версия истории

Перед масштабными правками через Gemini: Файл → История версий → Назвать текущую версию. Отмена действий Gemini иногда работает непредсказуемо — сохранённая версия гарантирует откат.

Формулы, макросы и VBA

ИИ пишет любую формулу по описанию задачи. Никакого Stack Overflow и часового гугления — просто объясните что нужно сделать.

🧮
ChatGPT / Claude / Gemini / Grok
Универсальный генератор формул
Freemium

Самый мощный способ — описать задачу на русском языке в чате. ChatGPT, Claude, Gemini и Grok понимают контекст финансовых задач и пишут XLOOKUP, LAMBDA, формулы массивов, а также VBA-макросы любой сложности. Grok — бесплатно, Gemini — бесплатно в Sheets.

  • XLOOKUP, INDEX+MATCH, SUMPRODUCT — по описанию задачи
  • Пишет и объясняет VBA-макросы — даже сложные
  • LAMBDA-функции с документацией
  • Отлаживает существующие формулы с ошибками
  • Объясняет чужой код VBA строка за строкой
Как работать — алгоритм

1. Откройте claude.ai или chatgpt.com в браузере рядом с Excel.
2. Опишите задачу: укажите название столбцов, откуда берутся данные, что должна вернуть формула.
3. Вставьте формулу в Excel → проверьте на 3–5 строках вручную.
4. Если не работает — скопируйте сообщение об ошибке (#VALUE!, #REF! и т.д.) обратно в чат.

Пример 1 — XLOOKUP с несколькими справочниками
Промпт
Напиши формулу Excel: ищет ИНН из ячейки A2 в столбце D на листе «Контрагенты», возвращает значение из столбца G. Если не найден — «Не найден». Используй XLOOKUP.
Пример 2 — VBA-макрос для консолидации
Промпт для VBA
Напиши VBA-макрос для Excel: проходит по всем листам книги (кроме «Сводная»), копирует данные из диапазона A2:F1000 (до первой пустой строки) на лист «Сводная». Перед копированием очищает «Сводная». Добавь столбец с именем листа-источника.
Пример 3 — Отладка ошибки
Промпт для отладки
Эта формула выдаёт ошибку #VALUE!: [вставь формулу]. Данные в столбце C — текстовые числа из выгрузки 1С. Исправь формулу и объясни в чём была проблема.

Сценарий: очистка данных из 1С

Выгрузки из 1С приходят с «мусором»: объединёнными ячейками, итоговыми строками посередине, текстовыми числами. Промпт ниже генерирует готовый макрос очистки.

Промпт (на русском)
Напиши VBA-макрос очистки данных для выгрузки из 1С: 1) Удали все строки где в столбце A слово «Итого» или «Всего»; 2) Преобразуй столбцы C:F из текста в числа (убери пробелы и замени запятую на точку); 3) Заполни пустые ячейки в столбце A значением из ячейки выше; 4) Удали полностью пустые строки. Код с комментариями на русском.
Как вставить VBA-макрос в Excel

Скопируйте код → в Excel нажмите Alt + F11 (открывается редактор VBA) → Insert → Module → вставьте код → F5 для запуска или Alt + F8 из Excel для выбора макроса.

Схема выгрузки данных из 1С в Excel

1
Прямая кнопка «Выгрузить в Excel»
Самый простой способ — встроена в большинство отчётов 1С. Минус: нужно делать вручную каждый раз. Подходит для разовых выгрузок.
2
Регламентное задание + планировщик Windows
Настраиваете автоматическую выгрузку в 1С через регламентные задания. Планировщик Windows запускает по расписанию — файл появляется в папке каждое утро.
3
OData-протокол → Power BI / Power Query
1С поддерживает OData — можно подключить Power Query напрямую без выгрузки файлов. Данные обновляются при открытии книги. ИИ напишет нужные M-трансформации.
4
Экстрактор 1С → BI (без программиста)
Специализированный коннектор — настраивается без разработчика. Поддерживает Yandex DataLens, Power BI, Google Data Studio. Данные обновляются автоматически.

Типичные проблемы выгрузок

Текстовые числа
Суммы приходят с пробелами и запятой как разделителем. Формулы не считают. → ИИ напишет макрос очистки за 30 секунд.
Итоговые строки посередине
«Итого по разделу» вставлены между данными — ломают сводные таблицы. → VBA по промпту удалит их автоматически.
Объединённые ячейки
Шапки и группы объединены — нельзя использовать как таблицу. → Макрос разъединит и заполнит пустые ячейки.
Разные форматы дат
1С выгружает даты как текст «01.03.2025» — Excel не сортирует, формулы не считают разницу. → Power Query или VBA приводит к типу Date.
